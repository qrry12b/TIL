# Javascript 올림, 내림, 반올림

## 올림

```
Math.ceil(x)
```

주어진 숫자(x)와 같거나 x보다 큰 가장 가까운 정수를 반환합니다.

## 내림

```
Math.floor(x)
```

주어진 숫자(x)와 같거나 x보다 작은 가장 가까운 정수를 반환합니다.

## 반올림

```
Math.round(x)
```
주어진 숫자(x)를 반올림한 수와 가장 가까운 정수를 반환합니다.
